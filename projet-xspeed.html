XSPEED

Contexte

Actuellement en cours de développement pour une sortie prévue vers septembre (sur PC et mobile), ce jeu de réaction rapide type rogue lite est un de mes projets personnels fruit de ma passion pour le développement appris en autodidacte.

Ce jeu est inspiré du type de jeu “Ice slider” originalement dans le registre puzzle et casse tête, que j’ai revisité dans un format rapide et nerveux, avec progression rogue lite (répétition de niveau avec bonus à débloquer) dont le but est d’atteindre des scores toujours plus hauts.

Essayer la Demo

Prototype Web Demo de gameplay “ice slider” en grille isométrique. Tentez d’atteindre les 10 000 points (astuce : dépassez les 100 de combo).

Commandes

↖ J : Haut gauche
| G ↗ : Haut droite
↙ N : Bas gauche
| B ↘ : Bas droite

Détails Techniques

  • Moteur : Godot 4.4
  • Version : Prototype web HTML
  • Hébergement : Netlify

Note : Version Démo Web très allégée avec de nombreuses fonctionnalités désactivées (niveaux, missions, achats, musiques...).

Réalisation & évolutions

Ce projet m'a permis d'appréhender la chaîne de production complète d'un jeu vidéo. En tant que développeur solo, j'ai dû alterner entre plusieurs rôles clés :

  • Game & Level Design
  • Programmation GDScript
  • UI/UX Design
  • Technical Art & Graphisme
  • Sound Design
  • Stratégie de déploiement
