Prototype : X-Speed Proto

Commandes

  • Z, Q, S, D : Déplacement
  • Espace : Saut
  • R : Recommencer le niveau

Détails Techniques

Ce prototype a été conçu pour tester la vélocité des déplacements et la gestion des collisions haute fréquence.

  • Moteur : Godot 4.x
  • Rendu : Compatibility Mode (OpenGL 3)
  • Hébergement : Netlify

Note : Si le jeu ne répond pas, cliquez une fois à l'intérieur du cadre pour activer les contrôles clavier.
